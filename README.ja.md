# 福井県観光ハッシュタグ別投稿数推移 on Instagram

このリポジトリは、福井県観光の代表的なInstagramハッシュタグの投稿数の推移を可視化するウェブアプリケーションです。

## デモ
https://code4fukui.github.io/fukui-kanko-hashtag/

## 機能
- 代表的な福井県観光ハッシュタグの投稿数を月別に表示
- ハッシュタグを選択して、投稿数の推移をグラフで確認できる

## 必要環境
特に必要ありません。ウェブブラウザがあれば利用できます。

## 使い方
ブラウザでデモページを開き、表示されているハッシュタグのドロップダウンメニューから興味のあるものを選択すると、投稿数の推移グラフが表示されます。

## データ・API
このアプリケーションは、福井県観光連盟が提供するオープンデータを利用しています。データは[オープンデータ標準利用規約](https://github.com/code4fukui/copyright-policy-jp/tree/main/OpenDataStandardTermsofUse)に基づいて利用しています。
