# fukui-kanko-hashtag

This project tracks the number of Instagram posts with various hashtags related to tourism in Fukui Prefecture, Japan.

## Demo
The demo is available at https://code4fukui.github.io/fukui-kanko-hashtag/

## Features
- Tracks the daily post count for several tourism-related hashtags in Fukui Prefecture
- Allows users to select a hashtag and view the daily post count trend
- Uses open data provided by Fukui Prefecture Tourism Federation

## Data / API
The data is sourced from CSV files in the repository, which are generated from the Instagram hashtag data.
